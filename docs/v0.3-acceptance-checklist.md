# OpenClaw Team Ops Console v0.3 Acceptance Checklist

状态：In Progress  
版本：v0.3 alpha  
适用仓库：`openclaw-team-ops-console`  
用途：用于 Codex 落地后的版本验收、回归检查、发布前评审

---

## 1. 验收目标

v0.3 alpha 的目标不是把项目做成控制平面，而是在已经成型的只读治理观察台基础上，继续把 **角色化 IA、指标可信度与评审叙事** 收口为更稳定的治理产品体验。

本次验收重点检查四件事：

1. 是否引入 **Target Registry**
2. 是否建立 **Evidence / Findings / Risks** 这条治理链路
3. 是否提供 **Recommendation（建议式运维）**
4. 是否在不破坏既有能力的前提下，继续保持 **严格只读边界**
