# v0.3 API Examples

The files in `docs/examples/api/` are illustrative JSON examples for the new governance endpoints.

## Included examples

- [targets.json](examples/api/targets.json)
- [findings.json](examples/api/findings.json)
- [evidence.json](examples/api/evidence.json)
- [risks-summary.json](examples/api/risks-summary.json)

## Notes

- the examples are read-only payloads
- they are shaped to match the current shared contracts
- they demonstrate mock, filesystem, and mixed-governance scenarios without implying any write capability
