# v0.2 DTO Changes

## New models

- `Target`
- `TargetSnapshotSummary`
- `Evidence`
- `Finding`
- `Recommendation`
- `RisksSummary`

## Extended models

- `TargetSnapshotSummary` now carries target-scoped inventory arrays
- response contracts now include target, evidence, finding, recommendation, and risks-summary responses

## Purpose

These changes allow the console to represent governance objects explicitly instead of inferring them from loose warning fields.
