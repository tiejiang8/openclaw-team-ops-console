```text
├── Targets

│   └── Finding Detail
├── Evidence
│   └── Evidence Detail
├── Agents

├── Sessions

├── Auth Profiles

```

## Layering

- Governance layer: `Targets`, `Risks`, `Findings`, `Evidence`
- Resource layer: `Agents`, `Workspaces`, `Sessions`, `Bindings`, `Auth Profiles`, `Topology`
- Overview bridges both layers with fleet-level metrics and drill-down entry points
