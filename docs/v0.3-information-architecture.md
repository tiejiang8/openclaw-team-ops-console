## Layering

- Governance layer: `Targets`, `Risks`, `Findings`, `Evidence`
- Resource layer: `Cron`, `Nodes`, `Agents`, `Workspaces`, `Sessions`, `Bindings`, `Auth Profiles`, `Topology`
- Runtime plane overlay: `Overview` runtime card + global runtime status bar
- Overview bridges governance, resource inventory, and runtime-plane visibility
